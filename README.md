# AIML_Real-Estate-Investment-Advisor

A Machine Learning-powered real estate analysis application that helps investors identify profitable properties and forecast future property values.
The system performs:


Classification to determine whether a property is a Good Investment or not.


Regression to predict the estimated property price after 5 years.


The project includes:


Data preprocessing and feature engineering


Exploratory Data Analysis (EDA)


Machine Learning model training and evaluation


Experiment tracking using MLflow


Interactive web application built with Streamlit


Business Value


Helps investors make data-driven real estate decisions


Supports identification of high-growth properties


Automates investment analysis for real estate platforms


Improves transparency and customer confidence with predictive insights


Tech Stack


Python


Pandas & NumPy


Scikit-learn


MLflow


Streamlit


Matplotlib & Seaborn


Features


Investment recommendation system


5-year price prediction


Interactive dashboard


ML experiment tracking


Real-time user input predictions
