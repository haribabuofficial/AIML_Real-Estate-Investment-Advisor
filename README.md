# Real Estate Investment Advisor

A Machine Learning-powered real estate analysis application that helps investors identify profitable properties and forecast future property values.

## Project Objectives
- Classify whether a property is a **Good Investment** or not.
- Predict the **estimated property price after 5 years** using Regression models.

## Features
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Classification and Regression models
- Investment recommendation system
- 5-year property price prediction
- Experiment tracking using MLflow
- Interactive web application using Streamlit

## Business Use Cases
- Help investors make data-driven real estate decisions
- Identify high-growth investment properties
- Automate investment analysis for real estate companies
- Improve customer trust with predictive insights

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- MLflow
- Streamlit
- Matplotlib
- Seaborn

## Machine Learning Tasks
### Classification
Predict whether a property is a:
- Good Investment
- Not a Good Investment

### Regression
Predict:
- Estimated property price after 5 years
